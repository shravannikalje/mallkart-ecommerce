# MallKart E-Commerce Platform

🛍️ A modern e-commerce platform with beautiful UI, digital clock, and shopping features.

## 📁 Project Structure

```
mallkart-ecommerce/
├── ui/
│   ├── index.html          # Modern homepage
│   ├── styles.css          # Responsive styling
│   └── script.js           # Interactive features
├── tools/
│   └── digital-clock/
│       ├── index.html      # Digital clock interface
│       ├── clock-styles.css # Clock styling
│       └── clock-script.js  # Clock functionality
└── README.md
```

## ✨ Features

### 🏪 E-Commerce UI (`ui/`)
- ✅ Modern responsive homepage
- ✅ Product grid with cards
- ✅ Shopping cart functionality
- ✅ Wishlist/Favorites system
- ✅ Search functionality
- ✅ Beautiful gradient design
- ✅ Mobile-first responsive design

### 🕐 Digital Clock (`tools/digital-clock/`)
- ✅ Real-time clock with 40+ timezones
- ✅ 12/24 hour format toggle
- ✅ Add/Remove custom timezones
- ✅ Search timezone functionality
- ✅ Preset timezone buttons
- ✅ UTC offset display
- ✅ Beautiful purple gradient UI
- ✅ Fully responsive design

## 🚀 How to Use

### View E-Commerce UI
1. Open `/ui/index.html` in your browser
2. Or access via GitHub Pages (if enabled)

### View Digital Clock
1. Open `/tools/digital-clock/index.html` in your browser
2. Select timezones from presets or add custom ones
3. Toggle between 12/24 hour format
4. Search for specific timezones
